Amazon-like storefront.

Roles and Actions
- `bamazonCustomer`
  - View Products for Sale
  - Purchase product
- `bamazonManager`
  - View Products for Sale
  - View Low Inventory
  - Add to Inventory
  - Add New Product
- `bamazonSupervisor`
  - View Product Sales by Department
  - Create New Department



## Tech Stack
- Node.js
- MySQL
